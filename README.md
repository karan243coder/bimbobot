## Environment Variables

<b><details><summary>Tap On Me For Environment Variable</summary>

- `BIMBO_API_ID` : Get From [my.telegram.org](https://my.telegram.org)
- `BIMBO_API_HASH` : Get From [my.telegram.org](https://my.telegram.org)
- `BIMBO_BOT_TOKEN` : Get From [BotFather](https://telegram.me/BotFather)
- `BIMBO_BOT_USERNAME` : Your Bot Username Without @
- `BIMBO_DB_URI` : Mongodb Database Url For Main Bot [Tutorial Watch Here](https://youtu.be/I36_OTWvT2w)
- `BIMBO_OWNER_ID` : It mean Admin/Owner Id For Broadcasting Message.
- `BIMBO_LOG_CHANNEL` : Log channel id start with -100xxxxxx
- `BIMBO_UPDATES_CHANNEL` : Update channel id start with -100xxxxxx
- `BIMBO` : True if you want token verification else False
- `BIMBO_URL` : Your Shortner Domain Url Without https://
- `BIMBO_API` : Your Shortner Api
- `BIMBO_TUTORIAL` : Your How To Open Link of Shortner

Heroku BuildPack :-
- `https://github.com/jonathanong/heroku-buildpack-ffmpeg-latest`

</details>
